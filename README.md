# Customer-Churn-Prediction-Dashboard
# 💡 Customer Churn Prediction Dashboard

A fully interactive **Machine Learning Dashboard** built with **Streamlit** to predict telecom customer churn using the **Gradient Boosting Classifier**.

## 🚀 Features
- User-friendly interface for churn prediction
- Real-time model inference
- Interactive visualizations using Plotly
- Gradient background and responsive UI design

## 🧠 Tech Stack
- **Frontend:** Streamlit, Plotly
- **Backend:** Scikit-learn, Pandas
- **Model:** Gradient Boosting Classifier
- **Language:** Python 3.10+
